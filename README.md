# SkyForge — Static Site (Variant B: Hero Background)

This package uses your supplied photo as a **hero-only background** with a soft overlay for readability.

## Publish on GitHub Pages
1. Create a new public repo (e.g., `skyforge`).
2. Upload these files to the repo root.
3. Enable **Settings → Pages** → Deploy from a branch → `main` / root.
4. (Optional) Add your custom domain under Pages and update DNS.

## Make the waitlist form live
- Remove `data-mode="demo"` in `index.html`.
- In `scripts.js`, add your Formspree/Formspark endpoint.
